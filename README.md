### Quick Start Guide

1.  Download the `SwarvePresence.zip` file from the link above.
2.  Unzip the file and run `SwarvePresence.exe`.
3.  Open the Discord desktop application.
4.  Enter your custom text in the **Secondary Line (State)** field.
5.  *Optional:* If you want the URL buttons to be visible, you **must** enter a valid **Asset Key** (a previously uploaded image) in the **Large Image Key** field.
6.  Click the **"Connect and Update Presence"** button.
